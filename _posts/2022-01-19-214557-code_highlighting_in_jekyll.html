Ah I see. Org doesn't work with highlight js because the latter needs pure text inside the <pre> block. How org does this is enclose each single word in the src block with a <span> and assign it a proper class. Then it'll use that class to tell the css how to render that single word.

See here for more information.

If I want to use highlight.js then will have to set (setq org-html-htmlize-output-type nil).
