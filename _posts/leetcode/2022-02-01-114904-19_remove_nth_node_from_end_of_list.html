1. Description

Given the head of a linked list, remove the nth node from the end of the list and return its head.

Constraints:

  1. The number of nodes in the list is sz.
  2. \(1 \leq sz \leq 30\).
  3. \(0 \leq Node.val \leq 100\).
  4. \(1 \leq n \leq sz\).

1.1. Examples:

Input: head = [1,2,3,4,5], n = 2
Output: [1,2,3,5]

Input: head = [1], n = 1
Output: []

Input: head = [1,2], n = 1
Output: [1]

2. Solution

2.1. Understanding the problem

From the provided code we can see it's a singly linked list.

We can use fast_node and fast_pos that will reach the end of the list, and slow_node and slow_pos that will be n+1 node(s) away from slow_pos. Slow node will be tracking the previous node of the node to be removed.

Given that the trivial case [1] and n=1 returns [], we have slow_node = None, slow_pos = -1.

2.2. Algorithm

  1. Initialise fast_node = head, fast_pos = 0, slow_node = None, slow_pos = -1
  2. while fast_node.next
    1. fast_node = fast_node.next, fast_pos += 1
    2. check
