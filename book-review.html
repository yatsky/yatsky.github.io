1.3.1 定义

  1. 阅读
    1. 第一种意义

      阅读只能增加资讯的东西。

    2. 第二种意义

      阅读比我们水平高上一截、能增进阅读者理解力的东西。

  2. 阅读艺术

    一个凭借头脑运作，除了玩味读物中的一些字句之外，不借助任何外部助力，以一己之力来提升自我的过程。只有在阅读难读的书时，可以寻找外界帮助。

  3. 阅读的层次
    • 基础阅读（elementary reading）层次，即识字的层次，了解一个字、词、句子说了什么。
      • 阅读准备阶段
      • 认字阶段
      • 词汇的增长以及对课文的运用
      • 几乎可以阅读所有读物，但是却不够老练
    • 检视阅读（inspectional reading）层次，特点在强调时间，在一定（短的）时间内，抓出一本书的重点。在这个层次的阅读上，你的目标是从表面去观察这本书，学习到光是书的表象所交给你的一切。典型问题是：“这本书在谈什么？”、“这本书的架构如何？”、“这本书包含哪些部分？”。
      • 有系统的略读或粗读，以发现这本书止不值得多花时间仔细阅读。帮助阅读者找到在分析阅读中一定要回答的问题，即找出本书的架构。
      • 粗浅的阅读，头一次面对一本难读的书的时候，从头到尾先读完一遍，碰到不动的地方不要停下来查询和思索（事实上，读这本书本身也应当这样做 — 我的评语）。帮助阅读者进入分析阅读中的第二个阶段，是阅读者想要了解全书内容的第一个必要步骤。
    • 分析阅读（analytical reading）层次，即全盘的阅读、完整的阅读，或者说是优质的阅读。
    • 主题阅读（syntopical reading），或是比较阅读（comparative reading）。在做主题阅读时，阅读者会读很多书，而不是一本书，并且列举出这些书之间相关之处，提出一个所有的书读谈到的主题。借助他所阅读的书籍，主题阅读者要能架构出一个可能在哪一本书里都没提过的主题分析。
  4. 自我要求的读者

    任何一个超出基础阅读的阅读层次，核心就在于你要努力提出问题，然后尽可能找出答案。阅读者的责任包括以下四个问题：

    1. 整体来说，这本书到底在谈什么？
    2. 作者细部说了什么，怎么说的？
    3. 这本书说得有道理吗？是全部有道理，还是部分有道理？
    4. 这本书跟你有什么关系？
  5. 不同的笔记种类
    1. 结构笔记（structural note-making），出现在检视阅读中
    2. 概念笔记（conceptual note-making），出现在分析阅读中
    3. 辩证笔记（dialectical note-making），出现在主题阅读中
  6. 辅助阅读
    1. 相关经验
    2. 其他的书
    3. 导论与摘要
    4. 工具书
    5. 搜索引擎（我自己加的，基本代替了工具书）
  7. 社会科学的定义（适用于本书目的的）
    1. 人类学、经济学、政治学、社会学
    2. 大部分有关法律、教育、公共行政的作品
    3. 一部分商业、社会服务的作品
    4. 大量的心理学作品
  8. 主题阅读
    1. 何为“同一个主题”？
      1. 单一的历史时期或是事件。
      2. 除1以外，很难定义“同一主题”，往往只能通过更多的阅读来判断所读的书是否为同一主题。
      3. 2似乎与“主题阅读”这一层次本身矛盾，即我们在开始阅读前就应当知道确认主题，但是这却是很难办到的。阅读和对主题的确认是相辅相成的。
      4. 我们可以利用“工具书”来帮助我们在某种程度上解决这一问题。

    主题阅读不是为了给我们的问题提供最终答案（尽管我们常常希望如此）。我们要做的是努力 面面俱到，又不预设立场 。

1.3.2 技巧

  1. 检视阅读

    这里所有的技巧读服务于一个目的，即了解一本书的重点、主题。

    1. 先看书名页，如果有序就看序。
    2. 研究目录页
    3. 索引页
    4. 出版者的介绍（到这一步为止，你应该能确定要不要真的略读这本书了）
    5. 根据你对这本书的目录和印象，挑几个看来跟主题息息相关的篇章来看，尤其仔细阅读篇章开头或结尾的摘要说明
    6. 把书打开，东翻翻西翻翻，念个一两段，但不要太多。就这样把全书翻过一遍，随时寻找主要论点的信号，留意主题的基本脉动。最重要的是，不要忽略一本书最后的两三页，作者往往在这里重新整理他认为既新又重要的观点。
  2. 分析阅读的规则（164页）
    1. 分析阅读的规则之阶段一（架构性）
      1. 对书籍进行分类
        1. 书籍分类的原则
          1. 实用性
            1. 严格来说，任何一本教我们如何生活，该做什么，不该做什么，同时说明做了会有什么奖赏，不做会有什么惩罚的伦理的书，都是一本实用的书。
          2. 理论性
            1. 除实用书籍以外的书籍都应当被归类为理论性的书籍。这类的书籍通常论述的是作者认为的事实及其理由，但不教导读者应当如何做。
      2. 使用一个单一的句子，或最多几句话来叙述整本书的内容
      3. 将书中重要篇章列举出来，说明它们如何按照顺序组成一个整体的架构（拟大纲）
      4. 找出作者想要问的问题及答案
    2. 分析阅读的规则之阶段二（诠释性）
      1. 找出共同的词义（coming to terms）

        找出重要单字，通过它们与作者达成共识。我们既要找到重要的单字，也要确认它们在使用时最精确的含义。

      2. 判断作者的主旨（将一本书中最重要的句子圈出来，找出其中的主旨）

        主旨声明的知识或观点。

      3. 从相关文句的关联中，设法找出一本书的基本论述
      4. 找出作者的答案

        确定作者已经解决了哪些问题，还有哪些是未解决的。在未解决的问题中，确定哪些是作者认为自己无法解决的问题。

    3. 分析阅读的规则之阶段三 — 训练读者受教、评论的能力（评论性）
      1. 在你说出“我同意”、“我不同意”，或“我暂缓评论”之前，你一定要能肯定地说：“我了解了。”
      2. 当你不同意作者的观点时，要理性地表达自己的意见，不要无理地辩驳或争论。

        （反面即是网络喷子）

      3. 化解争议 — 尊重知识与个人观点地不同，在作任何评断之前，都要找出理论基础。

        知识上的不同意见应当被看做是大体上可以解决的问题。

  3. 做笔记的方法
    1. 划线
    2. 双重划线
    3. 在空白处做星号或其他符号，但只需要在全书最重要的十几个地方即可。也可以加入书签。
    4. 在空白处为作者的议论编号。
    5. 在空白处记下其他的页码，强调在其他页码处有与此处相同、相关、不同的地方。
    6. 将关键字或句子圈出来。
    7. 在空白处记笔记。
  4. 为一本书拟大纲的技巧（基本上与 org mode 所推荐的组织整理方法无异）

    作者将全书分为 x 个部分，罗列出每个部分的观点。第一部分分为 y 个段落，每个段落的观点是什么。每个部分有 z 个阶段，罗列每个阶段的重点。

  5. 找出论述

    如果可以，找出书中说明重要论述的段落。但是，如果这个论述并没有这样表达出来，你就要去架构出来。你要从这一段或者那一段中挑选出句子来，然后整理出前后顺序的主旨，以及其组成的论述。 所有的论述都包含了一些声明（理由、结论），其次，论述有两种形式，一种是以一个或多个特殊的事实证明某种共通的概念（归纳法），另一种是以连串的通则来证明更进一步的共通概念（演绎法），最后，找出作者认为哪些事情是假设，哪些是能证实的或是有根据的，哪些是不需要证明的自明之理。

  6. 与作者辩论
    1. 理想化的辩论
      1. 心平气和地辩论
      2. 给出自己的前提或假设
      3. 尽力尝试不偏不倚（不让屁股决定脑袋）
    2. 如何在对立角度来评论一本书

      在读者说出：“我了解，但我不同意。”之后，他可以用以下的概念向作者说明：

      1. 你的知识不足（uninformed）
      2. 你的知识有错误（misinformed）
      3. 你不合逻辑－你的推论无法令人信服
      4. 你的分析不够完善
  7. 如何阅读实用型的书

    注意：此处的实用型书籍指的是指导人应当如何做才是好的的书籍。

    1. 实用书的种类
      1. 为了说明规则而写成的
      2. 为了阐述形成规则的原理而写成的
    2. 两个问题
      1. 作者的目的是什么（why）
      2. 他建议用什么方法达到这个目的？（how）
    3. 需要了解作者的性格、经历、时代背景

      因为实用型的书籍往往混杂了作者的雄辩和情绪。

    4. 阅读者的责任－其应当回答的四个问题
      1. 作者想要解决的问题是什么－他想要（让你）做什么？
      2. 作者的共识、主旨与论述
      3. 内容真实吗？
      4. 这本书与我何干？
  8. 如何阅读想象文学
    1. 不要抗拒想象文学带给你的影响力
    2. 在想象文学中，不要去找共识、主旨或论述
    3. 不要用适用于传递知识的、与真理一致的标准来批评小说
  9. 如何阅读故事、戏剧与诗（对此并不十分感兴趣故仅仅浏览了一下）
    1. 故事书

      一种诗满足我们个人潜意识需求的小说，一种是满足大多数人潜意识需求的小说。

      1. 读得快
      2. 全神贯注
  10. 如何阅读历史书
    1. “历史”的定义
      1. 就事实而言的历史（history as a fact）
      2. 就书写记录而言的历史（history as a written record of the facts）

      在这里使用的是第二种定义，因为事实是无法“阅读”的－就事实本身而言，没有问题可以提出，也不能增进我们的理解力。

    2. 阅读历史书要提出的问题
      1. 这本书谈了什么
      2. 作者细部说了什么，怎么说的？
      3. 批评历史
        1. 历史书不够真实
        2. 作者误用了资料
      4. 这本书跟你有什么关系？
  11. 如何阅读传记与自传（对此并不十分感兴趣故仅仅浏览了一下）
  12. 如何阅读关于当前的事件

    尤其要知道是谁在写这篇报导，了解他写作的心态。

    1. 这个作者想要证明什么？
    2. 他想要说服谁？
    3. 他具有的特殊知识、偏见、观念是什么？
    4. 他使用的特殊语言是什么？
    5. 他真的知道自己在说些什么吗？
  13. 如何阅读科学与数学

    仅限于传统中伟大的科学和数学的经典之作，以及现代科普著作。 阅读科学作品时有两个困难：

    1. 为了了解科学中归纳法的论点，你就必须了解科学家引以为理论基础的证据。而你可能对此一无所知。
    2. 数学的问题。

  15. 如何阅读经书
  16. 如何阅读社会科学
    1. 阅读社会科学的容易处

      我们每天接触到有关“社会”的话题，每天都有媒体在报道与“社会”相关的事件，我们对“社会”二字非常熟悉。

    2. 阅读社会科学的困难处

      社会科学不能数学化（not necessarily true nowadays）；其次，要说明定义比较困难；最后，社会科学的作品是混杂的，涉及到科学、哲学和历史，有时还会有虚构的成分。

    3. 阅读社会科学书籍需要“主题阅读”

      分析阅读中，阅读社会科学最困难的工作是回答“这本书是什么样的书？”，我们在阅读社会科学时，主要的着眼点应当在一个特殊的事件或问题上，而非一个特殊的作者或一本书。因此我们要引出“主题阅读”。

  17. 如何做主题阅读
    1. 准备工作
      1. 一定要知道：第一个特定的问题来说，所牵涉的不只是一本书而已。
      2. 要知道，总的来说，应该读哪些书？
      3. 一定要在确定书目后，进行“检视阅读”来了解每本书的主题、说明这是本什么样的书，并能列出大纲架构。
    2. 开始阅读

      注意：主题阅读有5个有顺序的步骤。

      1. 找到相关的章节－在主题阅读中，你应当把自己的主题放在第一位，把阅读单本的书放在叠认为。
      2. 引导作者与你达成共识－你需要为不同的作者建立起共识，强迫作者使用你的语言。
      3. 厘清问题－列出一些可以把我们想要了解的问题说得明白的问题，然后让那些作者来回答这些问题。作者们可能没有回答、或是回答不了这个问题，但我们可以尝试通过我们的思维找出间接的答案。
      4. 界定议题－如果一个问题被定义得很清楚，而且作者们也会用不同的方式来回答，那么这个议题就被定义出来了。我们应当根据作者提供的答案来进行分类。当两个作者都对一个问题有相当的了解，所做的回答却完全相反时，这就是一个非常有意义的议题。
      5. 分析讨论－把问题和议题按顺序排列，以求突出主题。比较有共通性的议题，要放在比较没有共通性的主题之前。

1.4 摘录

“学校是否有效地教导过学生如何阅读母语？可以说是，也可以说不是。到五六年级之前，整体来说，阅读是被有效地教导过，也学习过了。在这之前，我们发现阅读地学习曲线是稳定而普遍进步的，但是过了这一点之后，曲线就跌入了死寂的水平。这不是说以个人到了六年级就达到了个人学习能力的自然极限，因为证据一再显示，只要经过特殊的教导，成人及大一点的孩童，都能有显著的进步。同时，这也部表示大多数六年级学生在阅读各种实用书籍的时候，都已经有足够的理解能力。许许多多学生进入中学之后成绩很差，就是因为读不懂书中的意义。他们可以改进，他们也需要改进，淡他们就不这么做。” — James Mursel

太多的资讯就如同态少的资讯一样，都是一种对理解力的阻碍。 初学者的无知在于未学，而学者的无知在于学后。 — 蒙田 (也就是“学而不思则惘”) 如果一个人说他知道他在想些什么，却说不出来，通常是他并不知道自己在想些什么。 阅读的一部分本质就是“被困惑”，而且知道自己被困惑。 每位作者都有被瞎批评的痛苦经验。（让我想起了各类论坛的评论区） 把谈话当作是战争的人，要赢得战争就得为反对而反对，不论自己对错，都要反对成功。 当你的情绪很强烈时，你可能会认为自己很有道理。 不想被宣传所困惑，就得了解宣传得内容是什么。 阅读时越主动，就读得越好。
